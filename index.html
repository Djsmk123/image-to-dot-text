3

Write the scene

A scene is a short JSON timeline: who does what, when, for how long — max 5 seconds. Write it by hand or let an LLM write it for you.
🤖 How to generate a scene with an LLM
  1. Add your actors above — each chip's name is the id you'll refer to in the script.
  2. Copy the prompt below (it already lists your current actor names).
  3. Paste it into ChatGPT, Claude, or any LLM, and describe your scene in plain English at the end of it.
  4. Paste the JSON it replies with into the script box below, then click "Run scene".
4
